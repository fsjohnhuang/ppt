Data URI Scheme
HTTP URI Scheme
 http://www.github.com
data:image/png;base64,ivsdingasjifnsgjiansgsajdiangsd
data:[<mime type>][;charset=<charset>][;<encoding>],<encoded data>
①. data：协定名称；
②. [<mime type>]：可选项，数据类型（image/png、text/plain等）；
③. [;charset=<charset>]：可选项，字符集编码方式；
④. [;<encoding>]：数据编码方法(US-ASCII，base64等)；
⑤. <encoded data>：编码后的数据。
优点：
①. 减少图片、脚本和样式等资源的请求链接数
			
缺点：
①. 不会被浏览器缓存起来
②. 移动端性能较低
			
优化方案：
在css文件的background-image: url()中使用data uri scheme，既减少请求链接数，图片数据也随css文件被缓存起来。
			
浏览器支持：
①. 支持
Opera 7.2+ data URI 必须小于4100字符
IE8+ data URI必须小于32k（IE8不支持js的data URI）
Chrome、FF和Safari无限制
②. 不支持
